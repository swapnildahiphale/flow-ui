# Ren Okafor — working profile

Solo founder/developer. I build [Tideline](../projects/tideline-app/brief.md), a
small habit & focus analytics SaaS, and I maintain
[confetti-rs](../projects/confetti-rs/brief.md), an open-source Rust particle
library, on the side.

## How I work
- **Deep-work mornings.** My best code happens 8am–12pm. I batch meetings into
  the afternoon and keep them short — I genuinely dislike long meetings.
- **Ship small & often.** I'd rather merge five 30-line PRs than one 800-line
  one. Releases go out on Fridays (see [processes.md](processes.md)).
- **flow + Claude Code daily.** Every task starts as a flow task; most get a
  Claude session. The [weekly-review](../playbooks/weekly-review/brief.md)
  playbook runs every Friday to keep me honest about what shipped vs. stalled.

## Current focus
Getting Tideline from beta to GA. The gate is billing — see

polishing the streak dashboard and chasing a nasty timezone bug in the rollup
job.
